📋 Інструкція

  1. Завантажте шаблон Excel (кнопка "Завантажити шаблон")
  2. Заповніть дані у файлі
  3. Збережіть файл у форматі .xlsx
  4. Завантажте файл через форму вище
  5. Натисніть "Почати імпорт"

📌 Доступні колонки:

  • title * - Назва товару (обов'язково)
  • category - Категорія (створиться автоматично)
  • brand - Бренд (створиться автоматично)
  • price - Ціна закупки
  • retail_price - Роздрібна ціна
  • quantity - Кількість на складі
  • available - В наявності (Так/Ні)
  • description - Короткий опис
  • full_description - Повний опис
  • article - Артикул
  • code - Код товару
  • old_price - Стара ціна (для знижки)
  • discount - Відсоток знижки
  • warranty - Гарантія (місяців)
  • country - Країна виробництва
  • promotions - Акції (через кому)
  • attributes - Характеристики (у форматі JSON)
💡 Підказка: Колонки можуть бути українською або англійською мовою. Система автоматично розпізнає обидва варіанти.
